# Mohamad Oomer Habib Moussa — Portfolio

A single-page personal portfolio showcasing my Python, AI, and full-stack
projects. Built as a self-contained `index.html` with no build step or
dependencies.

🔗 **Live site:** enable GitHub Pages (Settings → Pages → Source: `main`) to
serve it at `https://oomer9030.github.io/Portfolio_Oomer/`.

## Highlights

- **Single file, zero dependencies** — pure HTML, CSS, and vanilla JavaScript.
- **Modern aurora background** — animated, blurred gradient blobs over a layered
  navy/teal base with a masked dot-grid overlay.
- **HARBOR teal palette** — `#0E1B2A` · `#224B46` · `#4FD1C5` · `#D9FFF8` · `#FFF7E6`.
- **Interactive project modals** — click **Details →** on any card for the full
  write-up, tech stack, and key features.
- **Responsive** — adapts from desktop to mobile with a collapsible nav.
